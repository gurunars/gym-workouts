# Low cal cycle = HIT

Bodyweight exercises are a priority.

# Routine

> 5 min cardio warm-up

> Wide grip pull ups mech. warm up (30x)

Superset {
    > Wide grip pull ups with own weight (max)
    Dropset {
        > Wide grip mech. pull ups (15)
        > Wide grip mech. pull ups (12)
        > Wide grip mech. pull ups (8)
    }
}

> Incline B.P. warm up (30x)

Superset {
    > Push ups (max) -> Knee push ups (max)
    Dropset {
        > Incline B.P. (15)
        > Incline B.P. (12)
        > Incline B.P. (8)
    }
}

Dropset {
    > Preacher curls (12)
    > Preacher curls (10)
    > Preacher curls (8)
}

Dropset {
    > Cable push downs (30)
    > Cable push downs (25)
    > Cable push downs (20)
}

Dropset {
    > Lateral raises (30)
    > Lateral raises (25)
    > Lateral raises (20)
}

Dropset {
    > Cable crossover (30)
    > Cable crossover (25)
    > Cable crossover (20)
}
